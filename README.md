# Aplikasi Kasir BBM

Aplikasi kasir sederhana berbasis PHP untuk menghitung jumlah liter BBM berdasarkan uang yang dibayarkan.

## Fitur

- Menampilkan daftar BBM
- Memilih jenis BBM
- Menghitung jumlah liter
- Menampilkan struk pembelian

## Teknologi

- PHP
